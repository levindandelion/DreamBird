# DreamBird
这是一个查看天气的微信小程序。

## 展示

![demo](demo.gif)
![codeImg](codeImg.jpg)

## 主要功能

* 查看实时天气、24小时天气预报、7天天气预报
* 查找城市、定位、配置用户城市数据、获取小程序码

## 数据

* 和风api 获取天气数据
* 微信小程序JavaScriptSDK 获取城市及定位
